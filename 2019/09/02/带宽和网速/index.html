带宽和网速

Band Width & Network speed

一、概念准备

1、带宽
在数字信号系统中，带宽用来标识通讯线路所能传送数据的能力，即在单位时间内通过网络中某一点的最高数据率，常用的单位为bps(bit per second,每秒多少比特)。
2、网速
网速一般是指终端上网时的速率，请求和返回数据所用时间的长短。网速大致分为：

  1. 接入速率：接入速率代表“接入线路”（从宽带接入服务提供商的宽带业务接入点BRAS到用户终端的这一段链路）能够达到的最高速率。接入速率越高，上网等待时间越短，用户感受越好。
  2. 上下行速率

  上行速率是指移动终端给基站发送信息时的数据传输速率。—上传速率。
  下行速率是指基站向移动终端发送信息时的传输速率—下载速率。
3、流量
流量是指终端对外发送数据与接收数据包的大小总和，单位采取1024进制，单位有B，KB,MB，GB。(B = Byte)

3、网络速度度量
  1 Byte = 8 bits
  1 Kb = 1 * 2^10 bits = 1024 bits
  1 KB = 1 * 2^10 Bytes = 1024 Bytes
  1 Mb = 1 * 2^10 Kb = 1024 kb
  1 MB = 1 * 2^10 KB = 1024 KB

二、结论

1、Mbps是带宽单位，在“Mbps” 单位中的“b”是指 bit(位)。带宽是按位来描述的。
2、MB/s为速度单位，在“MB/s”中的”B”是指Byte(字节)。因为数据是按字节传输的。
3、即：Mb/s 除以8就等于 MB/s，带宽除以8为理论网速值。
