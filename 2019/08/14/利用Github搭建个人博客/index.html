利用Github搭建个人博客

第一步 环境准备

一 安装Git工具

1、点击下载Git for Windows,安装完成后，可以看到

二 安装Node.js和配置好Node.js环境

1、安装Node.js和配置好Node.js环境。下载连接：Node.js
2、下载对应版本，默认安装环境变量自动配置。安装完成后打开cmd命令行界面，查看是否安装成功。
node -v

三 Github注册账号和新建仓库

1、建立仓库部分不再展示，建立好仓库后，点击仓库右上角settings按钮,跳转至仓库设置界面，向下滑动到GitHub Pages,点击勾选右侧出现一个网址。此时，我们的项目已经被部署到网络上，能够通过外网来访问此网页。

四 安装Hexo

1、Hexo主页：https://hexo.io/zh-cn/
